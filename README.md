## ğŸ— Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/campbellinoz/MyNovelCraft.git
   cd MyNovelCraft
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```


